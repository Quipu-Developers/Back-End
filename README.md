## environment
- Language: JavaScript
- Package Manager: npm 10.2.4
- Library: cors, express, mysql2

### local
- production - recommended
  
```bash
git clone https://github.com/Quipu-Developers/quipu-main-backend.git

cd quipu-main-backend

npm install --production

npm node server.js
```

 - develop

  ```bash
git clone https://github.com/Quipu-Developers/quipu-main-backend.git

cd quipu-main-backend

npm install

npm start
```
